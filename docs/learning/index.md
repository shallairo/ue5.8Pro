# Learning Log Index

Store chronological learning notes in this directory.

## Naming

Use:

```text
YYYY-MM-DD-HHMM-topic.md
```

Example:

```text
2026-06-03-1705-gpu-pass-demo-uav-crash.md
```

## Purpose

Each learning log should summarize:

- what we built or debugged,
- the key concepts behind it,
- the important pitfalls,
- the concrete takeaways worth remembering for future work.
